# KindHour — Supportive Time Widget

A minimalist Android widget that displays supportive time-based messages with calming visuals.

## Preview

Coming soon

## Features

- Time-based supportive messages
- Random message per time block
- Android home screen widget
- Offline-first
- Pastel + monochrome themes

## Tech Stack

- Flutter (Dart)
- Android Widget (home_widget plugin)
- Local JSON storage
